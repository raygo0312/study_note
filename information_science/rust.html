modintを使用したい場合はac-library-rsを使用する．1e9+7の場合はModInt1000000007．998244353の場合はModInt998244353を使用する．その他の場合はModIntを使用する．ここではModintとして説明する．

乗法の逆元

逆元を使用する場合は，inv()を使用する． 
n.inv(); // 1/n

階乗

階乗を使用する場合は，階乗を前計算しておく． 
let mut fact_table = vec![Mint::new(1); n + 1];
let mut factinv_table = vec![Mint::new(1); n + 1];
for i in 1..=n {
  fact_table[i] = fact_table[i - 1] * i;
}
factinv_table[n] = fact_table[n].inv();
for i in (1..=n).rev() {
  factinv_table[i - 1] = factinv_table[i] * i;
}

メモ化

memoise::memoise

静的に指定する場合 
#[memoise(n <= 100, m <= 50)]
fn comb(n: usize, k: usize) -> usize {
  if m == 0 {
    return 1;
  }
  if n == 0 {
    return 0;
  }
  comb(n - 1, k - 1) + comb(n - 1, k)
}
条件は挟むことも式で書くこともできる 
#[memoise(-100 <= n + k <= 100)]
動的に指定する場合 
#[memoise(n, k)]
BTreeMapでメモ化する場合 
#[memoise_map(n)]
