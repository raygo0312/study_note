定義

定義 definition とは物事に共通認識を持たせる行為とそれを述べたものである．新たな概念を定義するとき，その説明を定義文 definitional statement といい，定義文によって定義される概念を被定義項 defined term という．ある言葉の定義文で使われている概念をその言葉を使って定義することを

例えば，「正三角形とはすべての辺の長さが等しい三角形である」を定義という．
この文は正三角形ってどんなもの？ということを説明しているね

以下はすべて同じ意味である．

  • $\term{被定義項}$を$\term{説明文}$で定義する
  • $\term{被定義項}$とは$\term{説明文}$である．
  • $\term{被定義項}$を$\term{説明文}$とする．
  • $\term{説明文}$を$\term{被定義項}$という．
  • $\term{説明文}$を$\term{被定義項}$とおく．
  • $\term{説明文}$を$\term{被定義項}$と定める．

再帰的定義

再帰的定義 recursive definition とは説明文に被定義項が含むもののうち，被定義項を説明文で展開することを繰り返すうちに，被定義項が含まれなくなるものである．一方，そうでないものを循環定義 circular definition という．循環定義は論理的に間違いとされている．

結局，再帰的定義って何？
有名なのはこれ
$n!=n\times(n-1)!$
$0!=1$
基本的には，終わりを作るために複数の言葉で定義する．
実際に説明文で展開してみると……
$3!=3\times2!=3\times2\times1!=3\times2\times1\times0!=3\times2\times1\times1$
!の展開を繰り返すと，!がなくなったね
これを再帰的定義というよ．
あと，循環定義が何なのかわからない
例えば，林檎を「皮が赤い果実」と定義したとき，赤を「林檎の皮の色」のような定義する
この定義が循環定義
「林檎」は，「赤」で定義され，「赤」も，「林檎」で定義されている．結局「赤い」と「林檎」の正体がわからないままでしょ？
結局，林檎も赤もわからないままだからダメなんだね

数理論理学

数理論理学 mathematical logic は数学の形式化を目的とする学問である．形式化 formalization とは文字列で正確に表現することである．つまり，数学語のようなものを作ることを目的とする．数学を形式化することで，概念でしかなかった数学を目に見えるようにし，数学をものとして扱えるようになる．例えば，計算機に扱い方を教えることで数学を計算することができる．また，日本語などの自然言語での表現を避けることで厳密性を保つ．まず始めに論理の形式化を行う．論理の形式化の方法をまとめたものを論理体系 logical system という．論理体系で定められた文字列のことを論理式 logical formula という．論理体系は数学を記述する言語で，その文が論理式である．そのため，論理式のことを単に文 sentence ともいう．

論理体系

論理体系はいくつかあり，命題論理 propositional logic ，一階述語論理 first-order predicate logic ，二階述語論理 second-order predicate logic ，……などがある．二階以上の述語論理を高階述語論理 higher-order predicate logic とまとめたりする．命題論理から始まり，段々と表現力が高くなる．他にもたくさんの論理体系が考えられている．

なんで何種類もあるの？
言語には日本語と英語があるように，数学にも様々な言語があって，必要に応じて使い分けているんだ

論理体系の定義は2段階ある．それが統語論と意味論である．どちらも言語学の用語である．統語論 syntax は構文を指し，ど文字列を論理式とするかを定義する．意味論 semantics は文の意味を指し，論理式のふるまいや値を定義する．

意味論

意味論の定義の仕方はいくつか存在する．論理式に値を定める方法をモデル理論的意味論 model theoretic semantics という．これを単に意味論ということもある．一方，論理式のふるまいを定める方法を証明論的意味論 proof theoretic semantics という．これを単に証明論 proof theory ということもある．モデル理論的意味論は集合論の言葉を必要とするが，集合論では論理式で定義されるため，循環定義になってしまう．そのため，まずは証明論的意味論による定義で進めることにする．

集合論

集合論 set theory では論理式に使われる単語の形式化を目的とする．単語の振る舞いを論理式で表すことで形式化できる．素朴的には，集合 set とは物の集まりである．数学として扱う対象を表す概念であるから，論理式も集合として定義することが可能である．なので，言葉を用意した後に，論理式を集合として定義しなおし，さらに深い議論をすることが可能になる．
