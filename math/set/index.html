集合

集合 set とは数学的対象の集まりを1つの対象としたものであるが，厳密に定義すると集合の公理を満たしたものになる．この集合の公理は，今回はZFC公理系を採用し，紹介する．今ある完成されたものではなく，集合を構成する，ZFC公理系を作っていく流れで議論を進める．

集合に用意されている2項述語は$=$と$\in$である．$=$は相等関係を持つ．$\in$はZFC公理系で性質を定められている．$=$の公理を下にまとめると，

$=$の公理

反射律
$$\forall a,a=a$$
代入原理
$f$を任意の述語として， $$\forall ab,a=b\to\p{f\p{a}\iff\p{b}}$$

相等関係は同値関係の一種である．同値関係 equivalence relation とは反射律，対象律，推移律が成り立つ関係のことである．つまり，反射律と代入原理から，対象律と推移律を示すことができる．

$=$の定理

対称律
$$\forall ab,a=b\to b=a$$
推移律
$$\forall abc,a=b\land b=c\to a=c$$
$=$の対称律の証明

$$\forall ab,a=b\to b=a$$

証明 $f\p{x}\Def x=a$とする．反射律より，$a=a\Iff f\p{a}$は真である．$a=b$と代入原理より，$f\p{b}\Iff b=a$も真である．
$=$の推移律の証明

$$\forall abc,a=b\land b=c\to a=c$$

証明 $f\p{x}\Def x=c$とする．前提より$b=c\Iff f\p{b}$は真である．$a=b$と代入原理より，$f\p{a}\Iff a=c$も真である．
