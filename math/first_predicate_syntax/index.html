一階述語論理の統語論

ホーム -> 数学 -> 一階述語論理の統語論

項と述語

実は，命題論理の表現力だけで，数学の問題を表現するには不十分なんだ
もう一歩，原子命題を分解する必要があるんだ．

命題の分解

原子命題を分解してもっと表現力をあげることを目標とする．例として「7は素数である」を考える．これを「7」という項 term と「～は素数である」という述語 predicate に分解する．これを形式化して$P\paren{7}$とする．このとき，述語を命題関数 propositional function ，条件 condition ，性質 property ，概念 concept ，関係 relation といったりもする．

量化

$P\paren{7}$のほかにも，$P\paren{1},P\paren{2},P\paren{3},\cdots$などがあり様々な項を入れることが出来る．項の値が不特定のものを変数 variable という．一方，値が特定のものを 定数 constant という．変数$x$に対し「$x$は素数である」は命題であるとはいえない．なぜなら$x$が1のときは偽となるけど，2のときは真となり，真偽が定まらないからである．ここで，量化 quantification という表現を導入する．述語$F$に対し「すべての$x$について素数である」の「すべての」や「素数となる$x$が半分存在する」の「半分存在する」などが量化である．これによって，$x$が不特定でも，述語を満たす値の量によって真偽が定まるため，命題になる．量化を表す文字を量化子 quantifier という．

項の値として数でないものも考えることから変数のことを変項 variable といったり，定数のことを定項 constant といったりもするよ

束縛と自由

量化された項を束縛変数 bound variable といい，量化されていない項を自由変数 free variable という．自由変数は量化されていないため，特定の値をとると考える．つまり，自由変数は定数のことである．同様に，束縛変数は様々な値をとるため変数のことである．

一階述語論理

文字

命題を項と述語で表現する論理体系を一階述語論理 first-order predicate logic という．一階述語論理も命題論理と同じように，用語や文字の意味を説明したが，厳密な定義ではないことに注意してほしい．なにかよくわからない名前の付いた文字がルールに従ってならんでいるだけなのである．一階述語論理では次の文字を用いる．

一階述語論理で使う文字

項
$$\xi,\zeta,\eta,\cdots$$
述語
$$F,G,H,\cdots$$
真理関数
$$\rho,\cdots$$
量化子
$$\Lambda,\cdots$$

論理式

一階述語論理における論理式を次のように定義する．

定義

一階述語論理における論理式
  1. $F$が$n$項述語で，$\xi_1,\xi_2,\cdots,\xi_n$が項ならば，$F\paren{\xi_1,\xi_2,\cdots,\xi_n}$は論理式である．
  2. $\rho$が$n$項真理関数で，$\phi_1,\phi_2,\cdots,\phi_n$が論理式ならば，$\rho\paren{\phi_1,\phi_2,\cdots,\phi_n}$は論理式である．
  3. $\zeta$が項で$\phi$が論理式で$\Lambda$が量化子ならば，$\Lambda\zeta\paren{\phi}$は論理式である．
  4. (1),(2),(3)以外は論理式ではない．

自由変数がない論理式を閉論理式 closed formula という．
